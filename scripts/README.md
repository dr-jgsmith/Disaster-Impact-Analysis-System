# DIAS Scripts

This directory contains utility scripts for development, testing, and data generation.

## Scripts

- `generate_test_data.py` - Generate synthetic test data for DIAS testing
- `validate_test_data.py` - Validate generated test data
- `format.sh` - Run code formatters (Black, isort)
- `lint.sh` - Run linters and type checkers
- `docker-run.sh` - Build and run Docker containers

See individual script documentation for usage instructions.
